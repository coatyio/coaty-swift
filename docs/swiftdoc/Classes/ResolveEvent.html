ResolveEvent

class ResolveEvent<GenericCoatyObject> : CommunicationEvent<ResolveEventData<GenericCoatyObject>> where GenericCoatyObject : CoatyObject

ResolveEvent provides a generic implementation for all ResolveEvents. Note that this class should preferably initialized via its withObject() method.
