CommunicationManager

public class CommunicationManager<Family> : CocoaMQTTDelegate where Family : ObjectFamily

Manages a set of predefined communication events and event patterns to query, distribute, and share Coaty objects across decantralized application components using publish-subscribe on top of MQTT messaging.

Note

Because overriding declarations in extensions is not supported yet, we have to have all observe and publish methods inside this class.
