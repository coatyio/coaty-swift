DiscoverEvent

public class DiscoverEvent<GenericDiscover> : CommunicationEvent<DiscoverEventData<GenericDiscover>> where GenericDiscover : Discover

DiscoverEvent provides a generic implementation for all DiscoverEvents. Note that this class should preferably initialized via its withObject() method.
